# Domain language

### Artifact

Encompass various types of created work by both students and teachers.

### Checklist

List of boolean criteria against which the work is "evaluated". Could be done by teacher or students. Unfilled

### Evaluation

Completed checklist and artefact

### CanonicalChecklist

an "official" version of Evaluation which sets part of a standard (is BenchmarkEvaluation better?)

### Exemplar

Artifact and CanonicalChecklist

### ArtifactWithEvaluations

Artefact with Evaluation(s)
